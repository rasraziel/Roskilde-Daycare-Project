Information about Roskilde
---------------------------

Technical
---------------------------
In order for the mail service to work, the javax.mail.jar file has to be imported to the Project.
The file is available in this folder as well as video instructions to import it.
The .txt files have to be outside the src folder for the application to work without changing filepaths.
